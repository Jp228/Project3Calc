What is AAA testing?

A test is an operation for software programs to determine if they are correct or not. There are more than one type of tests for example; unit tests, integration tests and more but all of them report whether the program passes or fails. AAA pattern stands for Arrange, Act and Assert.

The first step of the AAA pattern is Arrange which inputs and targets. Arrange steps set up the test case. Act calls a function or method, calling a REST API, or interacting with a web page. Act targets on the behavior. Assertion is the last step of the test. Assert steps tell the user if the function or method passed or failed. Depending on the function, assertions may check simple things like numbers or string values. Assertion determines and tests if the function or program failed or passed.
